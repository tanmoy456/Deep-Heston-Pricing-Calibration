# Deep Neural Network–Based Option Pricing and Calibration of the Heston Model

This project implements a **deep learning approach** for option pricing and calibration of the **Heston stochastic volatility model**, inspired by the paper:  

> **“A Deep Neural Network Perspective on Pricing and Calibration in (Rough) Volatility Models”**  
> [Link to paper](https://papers.ssrn.com/sol3/papers.cfm?abstract_id=3322085)
